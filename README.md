# latitude_query_vault

### Ruby version
- Ruby v3.2.1
- Rails v7.0.4.3
- Yarn v1.22.19

### Configuration
- `bundle install`
- `yarn install`
- `rails db:create`

### Database creation
- `rails db:create`
- `rails db:migrate`

### Initialization
- `rails db:seed`
- `rails server`

### How to run unit tests
- `yarn test`
